#### 设计模式类型
- 创建型：主要用于描述如何创建对象
- 结构型：主要用于描述如何实现类或对象的组合
- 行为型：主要用于描述类或对象怎样交互以及怎样分配职责

#### 经验总结
1. 尽量使用组合替代继承；
2. 尽量关联抽象类，而不是具体类；
